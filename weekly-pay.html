주휴수당이란?

주휴수당은 근로자가 일정한 조건을 충족했을 때 유급휴일에 대해 받는 임금입니다. 쉽게 말해, 정해진 날에 성실히 일한 근로자에게 쉬는 날도 임금 일부를 보장하는 제도입니다.

  • 알바도 근로자라면 조건 충족 시 주휴수당 대상이 될 수 있습니다.
  • 월급제, 시급제, 일급제라는 형식보다 실제 근로관계와 조건이 중요합니다.
  • 급여에 이미 주휴수당이 포함되어 있는 계약인지도 확인해야 합니다.

받을 수 있는 조건

주휴수당 조건은 상황에 따라 해석이 필요하지만, 실무적으로는 아래 항목을 먼저 봅니다.

  • 1주 소정근로시간 15시간 이상인지 확인
  • 근로계약서나 구두 합의로 정한 근무일을 개근했는지 확인
  • 계속 근로가 예정되어 있는지 확인
  • 실제 근무시간과 휴게시간 기록이 남아 있는지 확인

“이번 주에 많이 일했으니 무조건 주휴수당이 나온다”기보다는, 원래 정해진 소정근로시간과 개근 여부를 함께 확인해야 합니다.

계산 공식

주휴시간 = 1주 소정근로시간 / 40 x 8시간
주휴수당 = 주휴시간 x 시급

주 40시간 이상 근무하는 경우에는 주휴시간을 보통 8시간으로 봅니다. 주 40시간 미만 알바는 비례 계산으로 예상할 수 있습니다.

사장님이 많이 하는 실수

  • 시급에 주휴수당이 포함되어 있다고 말하지만 계약서에는 명확히 적지 않는 경우
  • 주 15시간 이상 근무했는데도 “알바는 주휴수당이 없다”고 안내하는 경우
  • 휴게시간을 실제보다 길게 처리해 주 15시간 아래로 만드는 경우
  • 근무표가 계속 바뀌는데 주휴 조건을 한 번도 다시 계산하지 않는 경우

주휴수당을 확인할 때는 출퇴근 기록과 근로계약서의 소정근로시간을 같이 보면 분쟁 가능성을 줄일 수 있습니다.

주휴수당을 못 받는 경우

  • 1주 소정근로시간이 15시간 미만인 경우
  • 정해진 근무일에 결근해 개근 요건을 충족하지 못한 경우
  • 근로자가 아닌 위탁, 프리랜서 등으로 실제 계약관계가 다른 경우
  • 근무 종료, 퇴사 시점과 해당 주의 계속 근로 여부가 문제 되는 경우
