기본 사용 순서

1. 달력에서 근무일 선택

계산하려는 연도와 월을 확인한 뒤 실제 일한 날짜를 달력에서 선택합니다.

2. 시급, 출근/퇴근, 휴게시간 입력

시급을 입력하고 기본 출근·퇴근 시간과 휴게시간을 맞춥니다. 날짜별 시간이 다르면 상세화면에서 개별 수정할 수 있습니다.

3. 주휴수당/야간수당/연장수당 옵션 확인

사업장 규모와 근무 조건에 맞게 주휴수당, 야간수당, 연장수당, 휴일수당 옵션을 확인합니다.

4. 세금 공제 방식 선택

세금 없음, 3.3%, 4대보험 예상 공제, 직접 입력 중 실제 급여 처리 방식에 가까운 항목을 고릅니다.

5. 월급 계산하기 클릭

입력값을 확인한 뒤 월급 계산하기 버튼을 눌러 예상 총급여와 세후 금액을 확인합니다.

6. 결과 확인

기본급, 야간·연장·휴일수당, 주휴수당, 공제액을 나누어 보며 실제 급여명세서와 비교합니다.

7. 공유 링크 또는 저장 기능 사용

필요하면 공유 링크나 Excel 저장 기능으로 계산 결과를 보관하거나 전달합니다.

모바일/PC 모드 차이

PC에서는 날짜별 세부 확인이 쉬운 상세 화면으로, 모바일에서는 작은 화면에서 빠르게 입력할 수 있는 간편 화면으로 자동 시작됩니다.

상황별 입력 팁

카페 마감 알바퇴근이 22시를 넘으면 야간수당 옵션을 켜고 실제 야간 시간을 확인하세요.
주말 알바토·일만 일해도 주 15시간 이상이면 주휴수당 조건을 검토할 수 있습니다.
스케줄 변동 알바기본 시간과 다른 날짜는 날짜별 상세 화면에서 개별 수정하면 됩니다.

사용법 FAQ

날짜마다 출퇴근 시간이 다르면 어떻게 하나요?

먼저 기본 근무시간을 넣고, 다른 날짜만 선택해서 개별 출퇴근 시간과 휴게시간을 수정하면 됩니다.

계산 결과를 나중에 다시 볼 수 있나요?

공유 링크 또는 저장 기능을 사용하면 입력값을 다시 확인하기 쉽습니다. 민감한 개인정보는 입력하지 않는 것을 권장합니다.

5인 미만 사업장은 어떤 옵션을 봐야 하나요?

연장·야간·휴일 가산수당 적용이 달라질 수 있으므로 사업장 규모 옵션을 먼저 확인하세요.
