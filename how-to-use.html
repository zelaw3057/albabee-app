급여계산기 사용법

처음 온 사용자도 바로 계산할 수 있도록 입력 순서를 짧게 정리했습니다.

알바BEE는 근무일 선택 → 시간 입력 → 수당/공제 확인 → 계산 순서로 사용하면 됩니다.

1달력에서 근무일 선택
2시급·시간 입력
3월급 계산
계산기 열기

기본 사용 순서

  • 계산할 연도와 월을 확인합니다.
  • 달력에서 실제 일한 날짜를 선택합니다.
  • 시급, 출근/퇴근, 휴게시간을 입력합니다.
  • 주휴·야간·연장·휴일수당과 세금 방식을 확인합니다.
  • 계산 결과를 보고 필요하면 공유하거나 저장합니다.
